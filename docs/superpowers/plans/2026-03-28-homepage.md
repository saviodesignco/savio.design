## Task 3: Wire up the contact form (Formspree)

**Files:**
- Modify: `index.html` (JS section)

- [ ] **Step 1: Add Formspree form submission JS**

The form has `id="contact-form"`. Add this JS block at the bottom of `<body>`, after other scripts:

```html
<script>
  document.getElementById('contact-form').addEventListener('submit', async function(e) {
    e.preventDefault();

    const form    = e.target;
    const btn     = form.querySelector('[type="submit"]');
    const ENDPOINT = 'FORMSPREE_ENDPOINT_PLACEHOLDER';

    // If no endpoint configured, open mailto fallback
    if (!ENDPOINT || ENDPOINT === 'FORMSPREE_ENDPOINT_PLACEHOLDER') {
      const name    = form.querySelector('[name="name"]').value;
      const message = form.querySelector('[name="message"]').value;
      window.location.href = 'mailto:hector@savio.design'
        + '?subject=' + encodeURIComponent('Website inquiry from ' + name)
        + '&body='    + encodeURIComponent(message);
      return;
    }

    btn.textContent = 'Sending…';
    btn.disabled = true;

    try {
      const res  = await fetch(ENDPOINT, {
        method:  'POST',
        headers: { 'Content-Type': 'application/json', 'Accept': 'application/json' },
        body:    JSON.stringify({
          name:    form.querySelector('[name="name"]').value,
          email:   form.querySelector('[name="email"]').value,
          message: form.querySelector('[name="message"]').value,
        })
      });
      const data = await res.json();

      if (data.ok) {
        // Show confirmation, hide form
        form.style.display = 'none';
        document.getElementById('contact-confirmation').style.display = 'block';
      } else {
        btn.textContent = 'Send Message';
        btn.disabled = false;
        alert('Something went wrong. Please email hector@savio.design directly.');
      }
    } catch {
      btn.textContent = 'Send Message';
      btn.disabled = false;
      alert('Something went wrong. Please email hector@savio.design directly.');
    }
  });
</script>
```

## Task 4: Smooth scroll and nav active state

**Files:**
- Modify: `index.html` (JS section)

- [ ] **Step 1: Add smooth scroll for nav links**

Add this JS. Nav links use `href="#section-id"` — this handles the offset so the fixed nav doesn't cover the section heading:

```html
<script>
  // Smooth scroll with fixed-nav offset
  document.querySelectorAll('a[href^="#"]').forEach(link => {
    link.addEventListener('click', function(e) {
      const target = document.querySelector(this.getAttribute('href'));
      if (!target) return;
      e.preventDefault();
      const offset = 80; // nav height
      const top = target.getBoundingClientRect().top + window.scrollY - offset;
      window.scrollTo({ top, behavior: 'smooth' });
    });
  });

  // Nav scroll class
  window.addEventListener('scroll', () => {
    document.getElementById('nav').classList.toggle('scrolled', window.scrollY > 40);
  });
</script>
```

Click each nav link. Each section heading should appear just below the nav bar — not hidden behind it.
